# 📊 Pesquisa de Opinião — Agenda 8

Programa em Python que realiza uma pesquisa de opinião com 50 entrevistados sobre o atendimento da empresa TudoWeb.

## 🛠 Funcionalidades

- Coleta nome, idade e opinião de cada entrevistado
- Registra opiniões: Excelente, Bom ou Ruim
- Exibe o total de respostas Excelente e Ruim ao final

## 📁 Arquivos

- `jonatha_banim_das_neves_Ag8_DS_I.py` — código principal

## ▶ Como executar

```bash
python jonatha_banim_das_neves_Ag8_DS_I.py
```
